Struktur Database Apotek Naufal

🔑 = Primary Key (Kunci Utama)   |   🗝️ = Foreign Key (Kunci Relasi)   |   Label merah menunjukkan arah relasi tabel
# users
1 id 🔑 bigint
2 name string
3 email string
4 role enum
# categories
1 id 🔑 bigint
2 name string
3 slug string
4 is_active boolean
# medicines
1 id 🔑 bigint
2 category_id 🗝️ → categories.id bigint
3 name string
4 price decimal
5 stock int
# orders
1 id 🔑 bigint
2 order_number string
3 user_id 🗝️ → users.id bigint
4 status enum
5 total_amount decimal
# order_items
1 id 🔑 bigint
2 order_id 🗝️ → orders.id bigint
3 medicine_id 🗝️ → medicines.id bigint
4 quantity int
5 subtotal decimal
# prescriptions
1 id 🔑 bigint
2 user_id 🗝️ → users.id bigint
3 order_id 🗝️ → orders.id bigint
4 prescription_number string
5 status enum
6 verified_by 🗝️ → users.id bigint
