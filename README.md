## 💊 Apotek Naufal System

## 📌 Deskripsi

Apotek Naufal System adalah aplikasi berbasis web yang dikembangkan untuk membantu digitalisasi operasional apotek. Sistem ini memudahkan pengelolaan data obat, transaksi pembelian, serta laporan penjualan secara lebih efisien dan terstruktur.

## 🎯 Tujuan

- Mengurangi proses manual dalam pencatatan data

- Meningkatkan efisiensi operasional apotek

- Menyediakan laporan yang lebih akurat dan real-time

## ⚙️ Fitur Utama

🔐 Authentication (Login & Register)

👥 Multi Role (Admin, Pelanggan, Owner)

💊 Manajemen Data Obat

🛒 Keranjang & Checkout

💳 Pembayaran

📄 Upload Resep Dokter

📦 Manajemen Stok

📊 Report / Laporan
