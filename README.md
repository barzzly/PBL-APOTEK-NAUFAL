# 💊 Apotek Naufal System

---

## 📌 Deskripsi
**Apotek Naufal System** adalah aplikasi berbasis web yang dikembangkan untuk membantu digitalisasi operasional apotek. Sistem ini memudahkan pengelolaan data obat, transaksi pembelian, serta laporan penjualan secara lebih efisien, akurat, dan terstruktur.

---

## 🎯 Tujuan
- Mengurangi proses manual dalam pencatatan data  
- Meningkatkan efisiensi operasional apotek  
- Menyediakan laporan yang lebih akurat dan real-time  

---

## ⚙️ Fitur Utama
- 🔐 Authentication (Login & Register)  
- 👥 Multi Role (Admin, Pelanggan, Owner)  
- 💊 Manajemen Data Obat  
- 🛒 Keranjang & Checkout  
- 💳 Pembayaran  
- 📄 Upload Resep Dokter  
- 📦 Manajemen Stok Obat  
- 📊 Report / Laporan  

---

## 👥 Pengguna Sistem
- **Pelanggan** → mencari obat, upload resep, melakukan pembelian  
- **Admin** → mengelola data obat, memproses pesanan, update stok  
- **Owner** → melihat dashboard dan laporan

---

## 🔄 Alur Sistem
1. Pelanggan mencari obat atau upload resep  
2. Menambahkan ke keranjang dan melakukan checkout  
3. Melakukan pembayaran  
4. Admin memverifikasi dan memproses pesanan  
5. Sistem memperbarui stok dan menyimpan data transaksi  
6. Owner melihat laporan penjualan  

---

## 🚀 Manfaat
Sistem ini membantu apotek menjadi lebih modern, efisien, dan terintegrasi dalam pengelolaan data serta transaksi.
